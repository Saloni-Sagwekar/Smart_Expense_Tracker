Project Title

Smart Expense Tracker (Python)

Introduction

The Smart Expense Tracker is a console-based Python application that helps users manage their daily expenses. It allows users to add, view, search, and delete expenses while storing data permanently using file handling. The system also provides useful insights like category-wise spending and monthly reports.

Objectives
To manage daily expenses efficiently
To store expense data using file handling
To analyze spending patterns
To implement Python concepts like OOP and exception handling

Features
Add expense with category, amount, and date
View all saved expenses
Search expenses by category
Delete expense using unique ID
Calculate total spending
Category-wise analysis
Monthly report generation
Insights (monthly category analysis and spending trends)

Technologies Used
Python
Object-Oriented Programming (OOP)
File Handling
Exception Handling
datetime module

Concepts Used
Classes and Objects
Encapsulation
File read/write operations
Try-except error handling
Dictionary for data grouping

Files Used
Main Python file (Smart Expense Tracker.py)
expenses.txt (stores all expense data)

How to Run
Open the project in PyCharm or any Python IDE
Run the Python file
Use the menu options to manage expenses

Sample Input

Category: Food
Amount: 200
Date: 23-03-2026


Sample Output

ID: 1, Category: Food, Amount: 200.0, Date: 23-03-2026
